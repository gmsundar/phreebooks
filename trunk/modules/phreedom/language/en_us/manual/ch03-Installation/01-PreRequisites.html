Pre-requisites

Prior to running the setup script. the following items need to be verified:

System Requirements

During the setup script, PhreeBooks will inspect your system and let you know of any missing or downlevel pre-requisites. The setup script will not continue until these pre-requisites are met.

  1. Mysql Version: 4.0 or higher
  2. PHP Version: 5.0 and higher

Customization Requirements

  1. To load your own chart of accounts (versus one of the example charts), a file needs to be generated and put into the /modules/install/charts directory. Refer to the chart of accounts help file for format information.
