Running the Setup Script

Prior to setup you will want to ensure that the directory you are installing to is accessible from a web browser. You will also want to ensure that you have access to a database and have the database name, user name and password.

  • Unzip file into the desired directory on your server.
  • In a web browser, go to the location you unzipped the files to. You should receive a message stating that the configure.php was not found and a link to install the program. When you click the link you will be taken to the welcome page.

    • Welcome

  • You can read read through the information on the welcome page and then click continue.

    • License

  • Read and accept the user license and click continue.

    • Prerequisites

  • The installer will now check for the required files and directories. If there are any unmet dependencies that prevent continuing installation these will be shown in a warning box at the top of the page. You can click the 'more info' link next to unmet dependencies for information on how to fix the problem.
  • It is common for the MySQL database to be in red since the program may not yet have access to the database. This WIL NOT prevent installation.
  • You will also want to look at the bottom of the prerequisites page to ensure you have the correct file permissions set.
  • Once all of the required dependencies are met you can click install at the bottom of the page.

    • System Setup

  • You will now be asked the enter the directory or http(s) paths you want to use for various services. You can change these if you like but the defaults should work for most installations. You will also be asked if you want to enable SSL. Unless you know you have SSL working and want to use it you should leave this set to 'NO'.
  • Select 'Save Settings' once you have finished filling out all the boxes.

    • Database Setup

  • Database Type - Select the type of database you are using from the drop down menu.
  • Database Table - Prefix - In most cases this will be left blank. You can enter a prefix if you are using a single database for multiple applications.
  • Database Host - Default is localhost but you may enter another host name or IP address if database is not hosted locally.
  • Database Username - Username you entered when setting up database.
  • Database Password - Password you entered when setting up database.
  • Company Database Name - Enter the name of the database you setup. Contact the hosting provider if you are unsure.

    • Admin Setup

  • Enter your Administrator name, password and email. You can also choose whether you would like to check for updates when logging in.

    • Company Setup

  • Enter your company's information and click save.

    • Chart of Accounts

  • Choose a chart of accounts most appropriate for your company. Details of each chart can be viewed by selecting an account type and clicking 'view chart details' link. Save settings once you have made your selection.

    • Fiscal Year Setup

  • Choose the date and year you want your fiscal year to begin and save.

    • Account Defaults

  • PhreeBooks will attempt to determine the correct account numbers for the given items. This can be changed later but now is a good time to ensure that all of the account numbers are correct for the given items.

    • Finished

  • Your setup of PhreeBooks is now complete. This page will provide some information on further steps to help you configure PhreeBooks for your company's needs.
  • Make sure you set permissions on your configure.php, located in the /includes folder, back to read only.
  • Remove or rename your modules/install folder so that the system cannot be re-installed which would wipe out your existing database.
