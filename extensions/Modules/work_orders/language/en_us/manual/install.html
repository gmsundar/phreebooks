Installing/Removing the Work Order Module

The work order module is a part of the Phreedom Small Business Solution. The module can be installed and configured following a few easy steps.

Installation

  1. Download and unzip the work_order module from the Phreesoft webstore. (www.Phreesoft.com)
  2. Copy/upload the work_orders folder to the /path_to_phreedom/modules directory.
  3. Navigate to Company -> Module Administration.
  4. If Phreedom Module Administration permissions have been set to full, click on the Install button next to the Work Order title and description.
  5. The installation process will begin. If successful, the Work Order Module status will turn green and the revision information will appear in the Version column of the table.
  6. If a Properties icon appears, click on the icon and set any properties to configure the module.

Note: By default, your permissions for access to all the module features will be set to Full but no other user will be able to access the module until individual permissions have been set.

Uninstalling the Work Order Module

To uninstall  the Work Order module, follow the steps below.

NOTE: Removing the module will also remove the work order related database tables and module uploaded/create files in the my_files directory.

  1. Navigate to Company -> Module Administration.
  2. If Phreedom Administration permissions have been set to full access, Click on the Uninstall button next to the Work Order Module. Confirm the removal of the module and the process will begin.
  3. If successful, the green highlighting will be removed on the Work Order module line. Any error or informational messages will appear below the menu bar.
  4. OPTIONAL: The uninstall process will not remove the module files in the /modules/work_orders directory or any files, if they exist, in the my_files/DB_NAME/work_orders directory. These files must be removed from the server manually.
